Understanding Git.

Git is a free and open ended software which handles multiple projects of various size related to coding and website-making. It is a relatively simple skill to learn yet, it is extremely powerful, fast, and leaves a tiny digital footprint.


The way that git works can be understood as efficient and safe. When code is typed into a git repository, it can be in 3 different states:
  1. modified
  2. staged
  3. comitted

In a modified state, the contents of the file (the code) are being modified, but only in the working directory. These changes are yet to be saved and committed.


In a staged state, the changes in the files are moved to the staging area in the software. Git will see that these changes are to be a part of the next commit (save).

Finally, in the committed state, the files are saved, committed and sent to the website. This save will dsiplay your changes on the website or server. These changes are reversible by simply making a new change in the file and repeating this whole process all over again.

The Shaper.

The shaper is a tool used to shape wood and styrofoam. Using different drill heads can give varying results on the wooden board (I used the wooden board).

These are the steps taken to use the shaper in the lab:

  1. Rollup sleeves, tie hair, remove any wristwear to prevent drill injury
  2. Wear eye and ear protection
  3. Turn on the power
  4. Use the vacuum to get rid of the previous person's debris
  5. Take your plank and secure it to the workspace with 2 nails and double-sided tape (nails need to be flush with the wood)
  6. Use the shaper to scan the workspace as a new design.
  7. Load up your .svg file in the shaper
  8. Make sure to notify everyone in the lab that there will be a lot of noise


The shaper uses two drill bits. First, it uses the v-bit. The v-bit is a thin drill bit used for engraving or writing. The other tool is the 1/4" downcut flat end mill. This is used for making the pocket cuts in the board, or deep cutting.

FDM Printing.

This week, we focused on FDM printing which is 3D printing. 3D Priniting is a form of additive manufacturing. This means that we create an object by not taking away from the material being used.

An FDM (fused deposition modeling) printer is a 3D printer that is often used in early concept development and prototyping for medical device design. FDM refers to how the 3D model is formed. The material (molten plastic) is deposited by a three-axis system in single layers, and multiple layers fuse together to form the 3D model.



The process begins with a 3D CAD (computer-aided design) model. Slicing software divides the digital model into numerous slices (layers) and outputs a G-code file for the printer. The FDM printer heats solid plastic filament, and melts and extrudes it from a nozzle â€” layer upon layer â€” onto a build tray to form the 3D object. Each layer can be 0.1mm to 0.5mm thick based on the desired resolution.



The Prusa Slicer is a software which is used to transfer files into the .gcode format so that the files which we want to print can be converted into a language which the printer can understand and render a print.

The user must first load a .stl file onto the program. This is a type of 3D file of what the user might be trying to print. Once the file is loaded on, we can use the prusa slicer to change the size, infill, and adjust the supports required to print the object (if needed).

The objects size can be adjusted in mm or inches. This is if the object doesn't fit the printer bed size or to reduce the print time.

The infill stnads for the amount of internal strength your object will have. Infill determines the strength of your object as the printer will focus on how much PLA will be used to help form an internal structure, like a skeleton for the object.



Supports play a great role when we print an object which has overhangs. Over hangs are parts of a print that go beyond the centre of mass of the object. This means that the printer will be going against gravity to print this part with no support below it to print. Hence, the use of supports helps crete that extra bit of flooring for the overhang so that the print doesn't fail.

Fusion360.

Fusion360 is a 3D Modelling software from which you can generate .stl files by designing your own objects. In this week, I was tasked with designing a cup for the board which I had created in Week 2. For this, I had to revisit fusion360 to create this item.



I started off by taking the measurements of the gap which I had left for the cup to fit in. I created a circle to recreate the slot in which my cup will fit perfectly.


Once that was done, I lifted the cup by .25 inches as thats how deep the circular cut was when I used the shaper. I went on to lift the cup .25 inches, while leaving the outer circle of the cup to stay the same depth. This way it should slip right into the slot.



I then proceeded to add 2 more planes a few millimeters above each other to give my cup height. I made the middle plane ring a little smaller than the above ring so that I can get a vase-like figure for my cup. I was designing my cup to hold my desk stationery and thought that a vase figure would suit the purpose the best.



Once all the planes fit, I joined them to look like a vase. Now, the only thing left was to add a hollow space on the inside so that it could function as a cup. For that, I selected the topmost circular plane and deleted it to reveal a hollow inside of the cup. The cup was ready to be printed, but i hadf to add a form of design as a decoration.



As fusion360 didn't have the design I wanted, I saved my file as an .stl and exported it to Tnkercad. Once I was done adjusting and aligning my cup, I saved the .stl file and exported it to Prusa Slicer on the imac at the lab and printed it.
